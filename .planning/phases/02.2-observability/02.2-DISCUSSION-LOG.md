## time.Time{}.Unix() zero-value handling (2026-04-07 update)

| Option | Description | Selected |
|--------|-------------|----------|
| Use IsZero() check, output 0 | time.Time{}.Unix() = -62135596800 (not 0). Must check IsZero() explicitly before calling .Unix(). | ✓ |
| Output .Unix() directly | Wrong — would output large negative number for "never checked" state | |

**User's choice:** Use IsZero() check, output 0
**Notes:** Critical gotcha — time.Time{}.Unix() returns -62135596800, not 0. Implementation must use `if t := fn(); !t.IsZero() { ts = t.Unix() }` pattern.
